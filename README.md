# Storage

支持将其他系统云盘映射到本地与本地硬盘操作无差异

- 目前支持
  - `Minio` 支持
  - 阿里云`OSS` 未支持
  - 腾讯`Cos` 未支持

工具是基于[`dokan`](https://github.com/dokan-dev/dokan-dotnet) 实现的本地硬盘
